Delitos Informáticos

Las TIC como medio para cometer delitos

Dentro de esta clasificación de delitos, los delincuentes utilizan como instrumento a los diferentes medios tecnológicos para llevar a cabo acciones fraudulentas. Este tipo de delitos supone la premeditación con la conciencia absoluta de quien los realiza.

Las TIC como objetivo final en la ejecución de delitos

El fin principal de los delincuentes es dañar de manera física los dispositivos tecnológicos y todo tipo de hardware, afectando las computadoras personales de los usuarios. El principal delito asociado a esta clasificación es el robo de equipos de cómputo.

Delitos más comunes

  • Usar cajeros automáticos para robar los datos de acceso a las cuentas bancarias mediante la clonación y falsificación de tarjetas.
  • Usar medios de comunicación para comercializar copias ilegales de software con la que se violan los derechos de autor y se afecta la industria de desarrollo de software.
  • Usar el correo electrónico para mandar información o anuncios no deseados (spam) usando programas especializados en mandar correos indeseados a miles y hasta millones de usuarios de internet.

  • Usar claves de acceso ajenas para obtener información confidencial como lista de clientes de una empresa para venderla a la competencia. Esto se considera un delito grave.
  • Usar las computadoras para para apoderarse, utilizar o modificar datos personales de otra persona, apoderándose de su identidad.
  • Usar internet para estafar y chantajear con el pretexto de ofrecer empleos muy bien pagados haciendo que las personas envíen todos sus datos.
  • Usar páginas web para estafar a las personas, copiando, entre otras cosas páginas, las de los bancos, donde los usuarios hacen muchas transacciones.
  • Usar el correo electrónico para hacer espionaje informático sin que el usuario se dé cuenta. Los delincuentes utilizan correos electrónicos que se instalan en la computadora y roban información relevante.
  • Usar virus para atacar las computadoras. Hay diversos tipos de virus, veamos algunos:
    • Dialers: usan un marcado automático que se activa cuando el usuario se conecta a internet por teléfono.
    • Adware: recogen información de los hábitos de navegación. Se puede usar con fines publicitarios, ya que se ofrecen productos con base en las preferencias del usuario.
    • Programas de acceso remoto: permiten el acceso a la computadora del usuario para atacarla o alterarla. Afortunadamente son fácilmente reconocibles por los antivirus
    • Caballos de Troya: programas que provocan daños o ponen en peligro la seguridad del sistema de la computadora.
    • Gusanos: programas que además de provocar daño en los sistemas, se propagan a otras computadoras haciendo uso de la red, del correo electrónico, etc.
    • Programas espías (spyware): registran todo lo que se realiza una computadora. Se utiliza para obtener información confidencial o conocer cuál es el funcionamiento que un usuario le está dando a la máquina.

Los delitos presentados son muy comunes en la actualidad.

Delitos más comunes

A través de este tipo de delitos, las pérdidas para la víctima son menores en comparación con la clasificación anterior en la que los delincuentes utilizan las TIC como medio para cometer los delitos, ya que el acceso al hardware es limitado.

En las grandes compañías los centros de datos en los que se ubican los servidores principales cuentan con sistemas de seguridad física, por lo que las posibilidades de causar daño a las instalaciones son mínimas.

No podemos decir con exactitud el monto de dinero que se pierde por delitos relacionados ya que no hay información cierta, debido a que muchos de estos delitos no son denunciados por las personas afectadas, pero puede afirmarse que los daños son millonarios.
